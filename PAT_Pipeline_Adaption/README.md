## Setup

### ProB Model Checker
Please install the ProB model checker and place it properly under ./ProB folder.
Download of ProB is available [here](https://prob.hhu.de/w/index.php?title=Download).

### File Converter
Please place the file converter properly under ./fileconverter folder.

### Java
Please ensure Java 17 is properly installed before running the experiments.

## Running Experiments
Use the following command to run the experiments:
```
python3 pipeline.py
```

Example systems under experiment are stored in ./input-dataset.json.
